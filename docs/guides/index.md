# Guides

Step-by-step guides for deploying and integrating markdown-comment-sidecar.

## Deployment

- [Docker](docker.html) — containerize the comment server with a persistent volume
- [GitHub Pages](github-pages.html) — build docs in CI and deploy to GitHub Pages
- [Self-Hosting](self-hosting.html) — VPS, Railway, Render, and other platforms

## Integrations

Use markdown-comment-sidecar alongside the tools you already use:

- [VS Code](integrations/vscode.html) — local editing with live rebuild and browser preview
- [Obsidian](integrations/obsidian.html) — publish your vault with a comment layer
- [Notion Exports](integrations/notion.html) — add comments to pages exported from Notion
