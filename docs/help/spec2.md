# Getting Started

## Adding Comments

Select any text in the document to start a comment thread. A button will appear near your selection — click it to open the comment editor.

You can add comments in both **Preview** mode (rendered markdown) and **Markdown** mode (raw source).

---

## Replying to Threads

Click any comment card in the sidebar to open the full thread. Type your reply in the input at the bottom and press **Reply** or use `Cmd+Enter`.

---

## Resolving Threads

When a discussion is complete, resolve the thread using the **Resolve** button at the bottom of the thread view. You can optionally add a closing comment summarizing the outcome.

Resolved threads move to the **Resolved** tab in the sidebar and their highlights are removed from the document.

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Enter` | Submit comment or reply |
| `Escape` | Close modal or cancel |

---

## Themes

Use the theme selector in the top-right corner to switch between Classic, Dark, Sepia, Ocean, and Forest themes. Your preference is saved in the browser.
